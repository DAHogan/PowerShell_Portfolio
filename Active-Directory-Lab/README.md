# Active Directory Home Lab — SudoFixIt.local

## Overview
Brief 2–3 sentence description of what the project is.

## Environment
- Host: macOS Apple Silicon (UTM/QEMU)
- Domain Controller: Windows Server 2022
- Client: Windows 11 Pro (ARM64)
- Domain: SudoFixIt.local

## What I built
- Promoted Windows Server 2022 to a domain controller
- Created OUs, user accounts, and security groups
- Joined Windows 11 Pro to the SudoFixIt.local domain
- Managed user accounts (resets, lockouts, disable/enable)

## Skills demonstrated
| Skill | Tool |
|---|---|
| AD DS installation & configuration | Server Manager |
| Domain controller promotion | dcpromo wizard |
| OU & user management | ADUC (dsa.msc) |
| Domain join & troubleshooting | Windows Settings / DNS |
| PowerShell user automation | PowerShell 5.1 |
